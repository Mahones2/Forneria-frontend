# Configuración Local - Frontend

## Instalación Rápida

### Windows
```bash
setup.bat
```

### Mac/Linux
```bash
chmod +x setup.sh
./setup.sh
```

## Ejecutar Servidor

### Windows
```bash
run.bat
```

### Mac/Linux
```bash
chmod +x run.sh
./run.sh
```

## URL

Frontend: http://localhost:5173

## Requisitos

- Node.js instalado
- Backend ejecutándose en http://127.0.0.1:8000

## Notas

- Los scripts instalan automáticamente las dependencias y configuran el archivo `.env`
- El frontend se conecta automáticamente al backend local
